# Requirements
 - Python3
 - Ngrok (Snapd install recommended on linux)
 - Windows or Linux OS
# Features
 - Option to run with/without ngrok 
 - Easy server start with custom memory amount 
# How to use
 - To run  setup a minecraft server and accept eula.
 - Then just cd into the server directory and input "sudo python3 start.py" on linux, and click to run on windows
